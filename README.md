# 💻 Laptop Price Prediction Dashboard

## 🚀 Overview

This project predicts laptop prices based on specifications using machine learning models and provides an interactive comparison dashboard.

## 🎯 Problem Statement

Laptop prices vary based on multiple factors like RAM, CPU, storage, and brand. This project helps estimate fair prices based on these features.

## ⚙️ Models Used

* Linear Regression
* Random Forest
* XGBoost

## 📊 Results

* Best Model: XGBoost
* R² Score: ~0.79
* MAE: ~₹10000

## 🖥️ Features

* Predict price of a laptop
* Compare two laptops side-by-side
* Visual comparison chart
* Model performance metrics

## ▶️ Run Locally

pip install -r requirements.txt
streamlit run src/app.py

## 🧠 Train Model

python src/train_model.py

## 📁 Project Structure

* data/ → dataset
* models/ → trained model
* src/ → source code
